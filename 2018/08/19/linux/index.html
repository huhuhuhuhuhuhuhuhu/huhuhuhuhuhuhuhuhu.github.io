Linux常用命令总结1.0

1.当前目录、上层目录用什么表示？主目录用什么表示? 切换目录用什么命令
当前目录：
./
上层目录：
../
主目录：
~/
切换目录命令：
cd

2.怎么查看当前进程？怎么执行退出？怎么查看当前路径？
查看当前进程：
ps
执行退出：
exit
查看当前路径：
pwd

3.怎么清屏？怎么退出当前命令？怎么执行睡眠？怎么查看当前用户 id？查看指定帮助用什么命令？
清屏：
clear
退出当前命令：
ctrl+c 彻底退出
执行睡眠：
ctrl+z 挂起当前进程 fg 恢复后台
查看当前用户信息：
id：查看显示目前登陆账户的uid和gid及所属组及用户名
查看指定帮助：
man adduser 这个很全 而且有例子
–help 这个告诉你一些常用参数
info adduesr

4.Ls命令执行什么功能？常用选项？
功能：列出指定目录中的目录，以及文件
选项： a 所有文件 ；l 详细信息，包括大小字节数，可读可写可执行的权限等(ls -l = ll)

5.建立软链接(快捷方式)，以及硬链接的命令。
软链接：ln -s source link
硬链接：ln source link

6.目录创建用什么命令？创建文件用什么命令？复制文件用什么命令？
创建目录：
mkdir
创建文件：
典型的如 touch，vi/vim也可以创建文件，其实只要向一个不存在的文件输出，都会创建文件
复制文件：
cp：-r递归复制；-p保留文件权限

7.文件权限修改用什么命令？格式是怎么样的？
文件权限修改：
chmod
格式：
1、 chmod 751 file
2、 chmod u=rwx,g=rx,o=x file
3、chmod u+x g-w file

8.查看文件内容有哪些命令可以使用？
vi/vim：编辑方式查看，可修改
cat：显示全部文件内容
more：分页显示文件内容
less：与more相似，更好的是可以往前翻页
tail：仅查看尾部，还可以指定行数
head：仅查看头部,还可以指定行数

9.黑洞文件是哪个文件夹下的哪个命令？
/dev/null

10.移动文件用哪个命令？改名用哪个命令？
mv

11.删除文件用哪个命令？如果需要连目录及目录下文件一块删除呢？
rm：-r递归删除；-f强制删除

12.怎么使一个命令在后台运行?哪个命令专门用来查看后台任务?把后台任务调到前台执行使用什么命令?
&
job -l
fg

13.用什么命令对一个文件的内容进行统计？(行号、单词数、字节数)
wc：-l 统计行数；-w 统计字数：-c 统计字节数

14.终止进程用什么命令? 怎么查看系统支持的所有信号？
kill -9 pid(进程号)
kill -l

15.搜索文件用什么命令? 格式是怎么样的?
find /PATH 查找条件 [处理动作]
/PATH：查找路径，默认为当前目录
查找条件：常用-name filename(支持使用glob)

16.查看当前谁在使用该主机用什么命令? 查找自己所在的终端信息用什么命令?
tty：查看当前登陆终端
who：查看本机所有登陆
whoami：查看当前登陆用户；who am i：详细显示
w：查看登陆的用户、终端和正在执行的操作

17.用什么命令查看用过的命令列表?
history

18.使用什么命令查看磁盘使用空间？什么命令查看目录或文件大小？
df -h
内容：文件系统 容量 已用 可用 已用% 挂载点
du

19.使用什么命令查看 ip 地址及接口信息？
centos6：ifconfig
centos7：ip a

20.使用什么命令查看网络是否连通?
netstat
ping

21.通过什么命令查找执行命令?
which：只能查可执行文件
whereis：只能查二进制文件、说明文档，源文件等

22.进程状态有哪些？ps命令显示的信息中分别用什么符号表示？
运行态：R，处于执行状态
就绪态：ps信息中不显示，进程准备完成等待CPU调度
睡眠态：
可中断：S，等待某事件，而被挂起
不可中断：D，处于睡眠且不可被中断，指进程不响应异步信号
暂停态：T，暂停在内存中，无法自动被调度，需手动启动
僵死态：Z，父进程结束前，子进程不关闭，占用内存空间
终止态：X，进程结束或被终止，无法再执行
